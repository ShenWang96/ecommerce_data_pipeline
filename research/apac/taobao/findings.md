# Taobao/Tmall — 调研结论

**状态**: ❌ 不可用（需登录）

## 测试结果

| 端点 | 结果 |
|------|------|
| s.taobao.com 搜索页 | 显示"亲，请登录"，搜索结果显示"加载中..." |
| h5api.m.taobao.com 移动 API | RGV587_ERROR: "哎哟喂,被挤爆啦,请稍后重试!" |

## 原因

- Taobao 强制要求登录态才能查看搜索结果
- 移动端 API 需要 mtop 签名算法（私有实现）
- 即使 Playwright 渲染 JS，搜索结果区也只显示"加载中..."

## 可能方案

1. **淘宝开放平台 (TOP)** — 官方 API，但主要面向卖家/开发者，需要企业认证
2. **登录态维护** — 扫码登录后维持 cookie（但反自动化检测严格）
3. **替代** → 1688.com（阿里巴巴中国批发站）或 AliExpress（跨境）
